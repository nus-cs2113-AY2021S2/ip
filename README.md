# Duke User Guide

<br>

## Setting up Duke
1. Open Terminal(on mac) or Command Prompt(on Window)
2. Change directory to the folder where ip.jar is stored
3. Type in <code> java -jar ip.jar </code> to run Duke

<br>

## Features
1. Todo
2. Deadline
3. Event
4. List
5. Done
6. Delete
7. Find
8. Menu
9. Bye

<br>

## Feature 1: Todo
Adds a Todo task in your task list
